# Comparaison des modes CLI Copilot

<span class="badge-intermediate">Intermédiaire</span> <span class="badge-vscode">VS Code</span> <span class="badge-intellij">IntelliJ</span>

## Présentation

Ce tableau compare l'utilisation des modes CLI Copilot entre VS Code et IntelliJ IDEA.

| Mode | VS Code | IntelliJ |
|------|---------|----------|
| yolo | Oui (terminal intégré) | Oui (terminal intégré) |
| review | Oui | Oui |
| edit | Oui | Oui |
| chat | Oui | Oui |
| explain | Oui | Oui |
| test | Oui | Oui |

---

=== "VS Code"
    - Terminal intégré : toutes les commandes CLI Copilot sont utilisables
    - Intégration avec l'explorateur de fichiers
    - Possibilité d'utiliser les prompts `.prompt.md` en CLI

=== "IntelliJ IDEA"
    - Terminal intégré : toutes les commandes CLI Copilot sont utilisables
    - Intégration avec l'éditeur et les panels
    - Possibilité d'utiliser les prompts `.prompt.md` en CLI

---

!!! info "Compatibilité CLI"
    Les modes CLI Copilot sont identiques sur VS Code et IntelliJ dès lors que le terminal est configuré.
