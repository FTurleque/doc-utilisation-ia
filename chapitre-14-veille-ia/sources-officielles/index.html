    Sources Officielles & Changelogs¶

    Débutant

    Les sources les plus fiables sont les blogs et changelogs des éditeurs eux-mêmes. Cette page regroupe toutes les sources officielles à suivre, classées par éditeur.

    GitHub Copilot & GitHub¶

    La source numéro 1 pour tout ce qui concerne Copilot.

    Ressource URL Contenu Fréquence
    GitHub Blog github.blog Annonces produit, nouvelles fonctionnalités Copilot Hebdomadaire
    GitHub Changelog github.blog/changelog Toutes les modifications produit, filtrable par catégorie Quotidienne
    Copilot Changelog github.blog/changelog (filtre Copilot) Uniquement les changements Copilot Hebdomadaire
    GitHub Docs — Copilot docs.github.com/copilot Documentation officielle à jour Continue
    VS Code Release Notes code.visualstudio.com/updates Mises à jour mensuelles de VS Code (incluant Copilot) Mensuelle
    GitHub Universe githubuniverse.com Conférence annuelle — annonces majeures Copilot Annuelle (octobre)

    Astuce : le Changelog filtré

    Le GitHub Changelog filtré sur Copilot est la source la plus précise pour suivre chaque mise à jour de Copilot. Ajoute-le à ton lecteur RSS ou vérifie-le chaque semaine.

    Comment organiser ta veille¶

    Option 1 — Lecteur RSS (recommandé)¶

    La plupart de ces blogs ont un flux RSS. Utilise un lecteur comme Feedly, Inoreader ou Miniflux :

    # Flux RSS principaux
https://github.blog/changelog/feed/
https://openai.com/blog/rss.xml
https://www.anthropic.com/news/rss
https://code.visualstudio.com/feed.xml
https://blog.google/technology/ai/rss/

    Feedly est gratuit

    La version gratuite de Feedly supporte jusqu'à 100 sources et 3 dossiers. Largement suffisant pour démarrer.

    Option 2 — Alertes GitHub¶

    Pour suivre spécifiquement les releases d'outils :

    1. Va sur le repo GitHub de l'outil (ex. github.com/rtk-ai/rtk)
    2. Clique sur Watch → Custom → coche Releases
    3. Tu recevras un email à chaque nouvelle release

    Option 3 — Notifications VS Code¶

    VS Code affiche les notes de version automatiquement après chaque mise à jour. Lis-les — elles contiennent souvent les nouveautés Copilot.

    Résumé : les 5 sources essentielles¶

    Si tu ne devais en suivre que 5 :

    # Source Pourquoi
    1 GitHub Changelog (Copilot) Chaque mise à jour Copilot
    2 OpenAI Blog Nouveaux modèles derrière Copilot
    3 Anthropic News Modèle premium Copilot + Claude Code
    4 VS Code Release Notes Intégrations IDE mensuelles
    5 OWASP LLM Top 10 Sécurité IA à jour
